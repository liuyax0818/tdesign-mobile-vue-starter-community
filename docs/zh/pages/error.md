# 错误页 {#error}

## 设计思路 {#design}

基于 TDesign Mobile Vue 的 `t-result` 组件构建统一的错误页面模板。

该页面非常逻辑结构非常简单，开发者自行看代码即可，此处不再作多赘述。

## 模块结构 {#structure}

```
src/views/error/
├── 403/                 # 403 权限错误页面
├── 404/                 # 404 页面不存在错误
└── 500/                 # 500 服务器错误页面
```
